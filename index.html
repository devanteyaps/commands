GSPN card game

Full slash-command reference for the GSPN card system: pull packs, build your binder, trade, run the shop sim, and chase rarities—straight from the bot code. Each card includes a Usage line with example options (what to type or select in Discord).

GSPN Discord bot 89 commands

Start here

Wallet credits let you open shop packs, pay grading, and use the rarity upgrader. Cards are unique copies with a rarity tier (Common through Ultra Rare and beyond), special event tags, and sometimes effects during events.

Typical GSPN ladder: C < S < R < SR < SSR < SSR+ < UR < UR+ (plus event tiers like HOF / CHRC on special cards). Exact odds are pack- and event-dependent.

Flow

1) Earn credits (rewards, wheel, selling dupes, passive job).
2) /gspn_cards — open pack_open, check inventory / binder, view a card.
3) /shop under /gspn_cards buys packs; /gspn_market is player-to-player sales.
4) Use /gspn_cards extras help in Discord for the live interactive help panel.
